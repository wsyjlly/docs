

# SpringCloud

## SpringBoot与SpringCloud的版本对应

> SpringCloud包含了众多子项目，每一个子项目发布时间不一定一致，所以采用了伦敦地铁站为版本说明，而不是数字。并且从A开始是第一个发布版，B是第二个，以此类推。

大版本对应：

| SpringCloud版本     | SpringBoot版本 |
| ------------------- | -------------- |
| Greenwich(格林威治) | 2.1.x          |
| Finchley(芬奇利)    | 2.0.x          |
| Edgware(埃奇韦尔)   | 1.5.x          |
| Dalston(多尔斯顿)   | 1.5.x          |
| Camden(卡姆登)      | 1.4.x          |
| Brixton(布里克斯顿) | 1.3.x          |
| Angel(天使)         | 1.0.x          |
